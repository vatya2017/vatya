========================================
  여러분의 김비서를 만듭시다
========================================

3단계로 나만의 업무 비서를 만듭니다.
① 대시보드 만들기 → ② 다듬기 → ③ 명령어 등록


========================================
  ① 대시보드 만들기
========================================

아래 프롬프트를 Claude Code에 입력하세요:

────────────────────────────────

2-체험용-샘플 폴더에 있는 모든 데이터를 활용해서
나의 업무 대시보드를 HTML로 만들어줘.

이런 게 한눈에 보이면 좋겠어:
- 오늘 해야 할 일 (우선순위별, 체크박스로)
- 프로젝트별 진행률 (차트로)
- 이번 주 일정 (요일별)
- 매출 현황 요약

할 일을 직접 입력하고 추가할 수 있는 기능도 넣어줘.
완료한 일은 체크하면 줄이 그어지게 해줘.
디자인은 깔끔하고 다크모드로.

────────────────────────────────

완성되면 브라우저에서 열어보세요:
  Mac:     open 파일명.html
  Windows: start 파일명.html


========================================
  ② 다듬기 — 최소 3번은 수정해보세요
========================================

■ 디자인 수정:
- "색상을 좀 더 세련되게 바꿔줘"
- "차트를 파이차트 말고 막대 그래프로 바꿔줘"
- "폰트를 더 크게 해줘"

■ 기능 추가:
- "할 일을 직접 입력하고 추가할 수 있게 해줘"
- "완료한 일은 체크하면 줄이 그어지게 해줘"
- "모바일에서도 잘 보이게 반응형으로 해줘"

■ 내 것으로 만들기:
- "이름을 OOO으로 바꿔줘"
- "할 일 목록을 내 실제 할 일로 바꿔줘"
- "이번 주 일정을 내 진짜 일정으로 바꿔줘"


========================================
  ③ 명령어 등록 — /김비서 만들기
========================================

대시보드가 완성되면, 아래 프롬프트를 입력하세요:

────────────────────────────────

/김비서 라는 슬래시 명령어를 만들어줘.

이 명령어를 실행하면:
1. 2-체험용-샘플 폴더의 모든 데이터를 다시 읽어
2. 오늘 날짜 기준으로 할 일과 일정을 정리해
3. 대시보드 HTML을 최신 데이터로 업데이트해
4. 터미널에 오늘의 브리핑을 출력해:
   - 오늘 해야 할 일 (우선순위 순)
   - 오늘 일정
   - 미완료 업무 중 긴급한 것
   - 한마디 응원 메시지

────────────────────────────────

만들어지면 테스트해보세요:

  /김비서

→ 데이터를 읽고, 대시보드를 업데이트하고,
  오늘 브리핑을 정리해서 알려줍니다.
  매일 아침 이것만 치면 됩니다.


========================================
  집에 가서 발전시키기
========================================

오늘 만든 건 샘플 데이터 기반입니다.
집에 가서 본인 업무에 맞게 발전시켜보세요:

■ 데이터 교체:
- 본인 업무 목록, 프로젝트 현황, 일정으로 교체
- 엑셀이나 CSV 파일이 있으면 그걸로 연결

■ 기능 추가 아이디어:
- "메모장 기능도 넣어줘"
- "완료율 통계를 추가해줘"
- "지난주 대비 변화량도 보여줘"
- "북마크/즐겨찾기 링크 모음도 넣어줘"

■ 더 나아가기 (고급):
- Notion API 연동 → 노션 데이터를 자동으로 가져오기
- Google Calendar 연동 → 진짜 일정 자동 반영
- Gmail 연동 → 메일 요약까지 한 화면에
- Vercel 배포 → 어디서든 대시보드 접속

이런 것들은 Claude Code한테 물어보면 다 알려줍니다.
모르면 물어보세요. 하나씩 추가하다 보면
진짜 나만의 업무 비서가 완성됩니다.


========================================
  다른 아이디어도 좋습니다
========================================

위 예시 말고 본인이 실제로 필요한 도구를 만들어도 됩니다.

예시:
- "회의록에서 액션 아이템 뽑아서 칸반보드로 만들어줘"
- "매출 데이터로 월별 트렌드 리포트를 만들어줘"
- "주간일정을 캘린더 형태로 시각화해줘"
- "견적 계산기를 만들어줘"

핵심은:
- 무엇을 만들지
- 어떤 기능이 필요한지
- 어떤 느낌이면 좋겠는지

코드를 몰라도 됩니다. 말로 설명하면 Claude가 만들어줍니다.
