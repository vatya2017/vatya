========================================
  타이포그래피 영상 만들기 (Remotion)
========================================


■ 1단계: Remotion 프로젝트 생성
────────────────────────────────

터미널에서 아래 명령어를 입력하세요:

  npx create-video@latest

- 프로젝트 이름을 물어보면 아무거나 입력
- 설치가 끝나면 생성된 폴더로 이동:

  cd 프로젝트이름


■ 2단계: Claude Code 시작
────────────────────────────────

  claude


■ 3단계: 프롬프트 입력
────────────────────────────────

아래를 참고해서 본인 이름으로 바꿔 입력하세요:

"제 이름은 서만호입니다. My name is Seo Manho.
이 한글과 영어를 가지고 트렌디한 타이포그래피 영상을
만들어줘. 글자가 하나씩 떨어지는 효과로 해줘.
배경은 검정, 글자는 흰색, 폰트는 크고 굵게.
세로 영상(1080x1920)으로 해줘."


■ 4단계: 미리보기
────────────────────────────────

  npm run studio

- 브라우저에서 영상을 미리 확인할 수 있습니다
- 마음에 안 들면 Claude Code에 수정 요청


■ 5단계: 렌더링 (MP4 저장)
────────────────────────────────

  npx remotion render

- out/ 폴더에 mp4 파일이 생성됩니다


■ 효과 요청 예시
────────────────────────────────

- "글자가 줌인하면서 등장하게 해줘"
- "색상 그라데이션 효과 넣어줘"
- "글자가 회전하면서 나타나게"
- "스프링처럼 통통 튀는 효과"


■ 수정 요청 예시
────────────────────────────────

- "글자 색상을 골드로 바꿔줘"
- "애니메이션 속도를 좀 더 느리게"
- "배경에 파티클 효과 넣어줘"
- "마지막에 페이드아웃 효과 추가해줘"


■ 시간이 오래 걸리면
────────────────────────────────

Remotion 설치에 시간이 걸릴 수 있습니다.
안 되시면 다음 실습으로 넘어가셔도 됩니다!
