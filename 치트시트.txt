========================================
  클로드 코드 실전 워크숍 - 치트시트
========================================


■ 터미널 기본 명령어
────────────────────────────────
cd 폴더명          폴더로 이동
cd ..              상위 폴더로 이동
ls                 파일 목록 보기
pwd                현재 위치 확인
mkdir 폴더명       새 폴더 만들기
open 파일명        파일 열기 (Mac)
start 파일명       파일 열기 (Windows)
cat 파일명         파일 내용 보기
clear              화면 지우기


■ Claude Code 명령어
────────────────────────────────
claude             Claude Code 시작 (대화형)
claude -p "..."    명령 하나만 실행하고 종료
claude --continue  마지막 대화 이어하기
claude --resume    특정 세션 이어하기


■ Claude Code 안에서 쓰는 명령어
────────────────────────────────
/help              도움말
/clear             대화 초기화
/status            현재 상태 확인
Ctrl+C             현재 작업 중단
Ctrl+D             Claude Code 종료


■ 자주 쓰는 프롬프트 패턴
────────────────────────────────
"이 파일 읽고 요약해줘"
"이 데이터 분석해줘"
"HTML로 OOO 만들어줘"
"여기에 OO 기능 추가해줘"
"디자인을 OO하게 바꿔줘"
"이걸 CSV로 다운로드 되게 해줘"


■ 파이프 (다른 명령어 결과를 Claude에게 넘기기)
────────────────────────────────
cat 파일.txt | claude -p "요약해줘"
cat 데이터.csv | claude -p "분석해줘"


■ 브라우저에서 결과 확인하기
────────────────────────────────
Mac:     open index.html
Windows: start index.html


■ 문제가 생기면
────────────────────────────────
1. Ctrl+C로 중단하고 다시 시도
2. claude 를 다시 입력해서 새로 시작
3. 손을 들어서 도움 요청!
