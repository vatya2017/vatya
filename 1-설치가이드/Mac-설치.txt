========================================
  Claude Code 설치 - Mac
========================================

■ 1단계: 터미널 열기
- Cmd + Space → "터미널" 검색 → 엔터

■ 2단계: Claude Code 설치
- 아래 명령어를 복사해서 붙여넣고 엔터:

  curl -fsSL https://claude.ai/install.sh | bash

- 설치가 끝나면 터미널을 껐다 다시 열어주세요

■ 3단계: 작업 폴더로 이동
  cd ~/Downloads/워크숍-실습

■ 4단계: Claude Code 시작
  claude

■ 5단계: 로그인
- 화면에 나오는 안내를 따라 Claude 계정으로 로그인

■ 6단계: 확인
- "안녕 나는 OOO이야" 라고 쳐보세요
- 답변이 오면 성공입니다!

■ 안 되면?
- 손을 들어주세요!
