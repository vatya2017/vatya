========================================
  Claude Code 설치 - Windows
========================================

■ 사전 준비
- Git for Windows가 필요합니다
- 아직 없으면: https://git-scm.com/downloads/win 에서 설치

■ 1단계: PowerShell 열기
- 시작 메뉴 → "PowerShell" 검색 → 엔터

■ 2단계: Claude Code 설치
- 아래 명령어를 복사해서 붙여넣고 엔터:

  irm https://claude.ai/install.ps1 | iex

- 설치가 끝나면 PowerShell을 껐다 다시 열어주세요

■ 3단계: 작업 폴더로 이동
  cd ~\Downloads\워크숍-실습

■ 4단계: Claude Code 시작
  claude

■ 5단계: 로그인
- 화면에 나오는 안내를 따라 Claude 계정으로 로그인

■ 6단계: 확인
- "안녕 나는 OOO이야" 라고 쳐보세요
- 답변이 오면 성공입니다!

■ 안 되면?
- 손을 들어주세요!
