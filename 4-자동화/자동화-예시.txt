========================================
  진짜 자동화란
========================================


■ claude -p는 자동화가 아닙니다
────────────────────────────────

claude -p "오늘 할 일 정리해줘"

이건 "단축 실행"입니다.
결국 내가 명령을 쳐야 실행되니까, 진짜 자동화는 아닙니다.


■ 진짜 자동화 도구들 (컴퓨터 꺼도 돌아감)
────────────────────────────────

업무 자동화 플랫폼:
- Zapier    — 가장 쉬움. 7,000개+ 앱 연동. 무료 100회/월
- Make      — 시각적 빌더. 가성비 좋음. 무료 1,000회/월
- n8n       — 셀프호스팅하면 무료 무제한. 자유도 최고

블로그 자동 작성:
- 가제트(Gazet)   — 키워드만 넣으면 SEO 블로그 글 자동 생성 (월 9,900원~)
- Koala Writer    — WordPress 자동 발행 ($9/월~)

뉴스/보고서 자동화:
- Perplexity  — AI 뉴스 요약 + 보고서 자동 생성 ($20/월)
- Feedly      — RSS 뉴스 수집 + AI 필터링 ($6.99/월~)
- Gemini      — Gmail, 캘린더, 뉴스 등 Google 연동


■ Claude Code의 강점
────────────────────────────────

위 서비스들은 편하지만 월 구독료가 나갑니다.
Claude Code는 이런 자동화 시스템 자체를 직접 만들 수 있습니다.

예시:
- "매일 아침 뉴스 크롤링해서 보고서 만드는 스크립트 짜줘"
- "블로그 글 생성하고 WordPress에 올리는 도구 만들어줘"
- "엑셀 파일 들어오면 자동으로 분석 보고서 만드는 프로그램 만들어줘"

→ 남이 만든 서비스에 월 구독료를 내는 대신
  내가 필요한 기능만 딱 만들어서 쓸 수 있습니다.
→ 오늘 배운 걸 기반으로 하나씩 확장해보세요.


========================================
  세션 관리 — 쓸수록 똑똑해지는 비결
========================================


■ CLAUDE.md
────────────────────────────────

프로젝트 폴더에 CLAUDE.md 파일을 만들어두면
새 세션을 열 때마다 자동으로 읽어갑니다.

적어두면 좋은 것:
- "한국어로 대답해"
- "이 프로젝트는 ○○이야"
- "파일 구조는 이렇게 되어 있어"


■ memory (자동 기억)
────────────────────────────────

Claude가 작업하면서 배운 것을 자동으로 기억합니다.
같은 프로젝트에서 쓸수록 맥락을 더 잘 이해합니다.


■ 세션 이어하기
────────────────────────────────

/continue     마지막 대화 이어서 하기
/resume       특정 세션 다시 열기

대화가 길어지면 새 세션에서 시작하는 게 더 빠릅니다.
CLAUDE.md와 메모리 덕분에 맥락이 유지되니까 걱정 마세요.
